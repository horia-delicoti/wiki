---
title: Reconnaissance Tools
description: Tools for network and web reconnaissance, enumeration, and information gathering.
---

Reconnaissance is the first phase of penetration testing. These tools help identify targets, discover hosts, services, and gather intelligence about systems.

### Web Scanning and Enumeration

- [Gobuster](/security/tools/reconnaissance/gobuster) - Fast directory and DNS brute-forcing tool.
- [Nikto](/security/tools/reconnaissance/nikto) - Web server scanner for vulnerabilities and misconfigurations.
- [WPScan](/security/tools/reconnaissance/wpscan) - WordPress security scanner.
- [Wappalyzer](/security/tools/reconnaissance/wappalyzer) - Technology profiler that identifies software used on websites.
- [WhatWeb](/security/tools/reconnaissance/whatweb) - Web technology identifier and fingerprinter.

### Related

- [Nmap](/networking/nmap) - Network discovery and port scanning.
- [Netdiscover](/networking/netdiscover) - ARP reconnaissance for discovering hosts.
